Notes to reviewer ...

    This app was created with Android Studio 3.1.3. Testing was completed on a Sony E5823 with Android 7.1.1 (API 25). Images were formatted to multiple densities using the Android Drawable Importer plugin. A custom object (Song) fills an ArrayList, a custom ArrayAdapter (SongAdapter) inflates views of these objects to populate a ListView. The custom objects passed between activities at runtime by implementing the Serializable interface.
